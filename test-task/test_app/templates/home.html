Inroduction

This project is based on Django framework with SQLite as datastore. It's intended to upload csv files. After uploading these files are parsed and stored into database.


Usage

Before starting the server the following preparations should be done:

1. Create virtual environment and activate it:

2. Go to project folder and install requirements:

pip install -r requirements.txt


Since internal sqlite database was initialized, just run the following command to start the server:

python manage.py runserver <host>:<port>
